## 📝 文档规范

### 命名规则

- **修复日志**: `模块名-修复日志-日期.md`（如：`实时流处理模块-修复日志-20260507.md`）
- **分析报告**: `模块名-模块分析报告.md`
- **设计文档**: `模块名-xxx设计.md`
- **配置文档**: `模块名-xxx配置.md`

### 版本管理

文档版本随代码版本同步更新，重要变更应记录在对应模块的修复日志中。
