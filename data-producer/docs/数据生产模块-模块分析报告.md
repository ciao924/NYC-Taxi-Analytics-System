
# Data Producer 模块分析报告

## 1. 模块概述

Data Producer 是纽约市出租车数据分析系统的**数据生产与模拟模块**，负责将历史 Parquet 格式的出租车数据以可控速率写入 Kafka 主题，模拟真实的实时数据流。该模块为整个实时数据处理链路提供数据源支撑。

### 1.1 核心功能

| 功能 | 描述 | 状态 |
|------|------|------|
| **多数据源支持** | 支持 Green 和 Yellow 两种出租车数据 | ✅ |
| **速率控制** | 可配置的消息发送速率，模拟真实流量 | ✅ |
| **断点续传** | 自动保存发送进度，重启后从断点继续 | ✅ |
| **数据校验** | 内置数据校验机制，过滤无效记录 | ✅ |
| **优雅退出** | 支持 Ctrl+C 中断，确保数据不丢失 | ✅ |
| **日志监控** | 完善的日志记录，支持控制台+文件输出 | ✅ |

### 1.2 数据流

```
Parquet 文件 → 数据读取 → 数据校验 → 速率控制 → Kafka 发送
     ↓              ↓          ↓          ↓          ↓
  数据源        Pandas    有效性检查    速率限制    目标Topic
```

## 3. 核心实现分析

### 3.1 主程序入口 - kafka_producer.py

#### 3.1.1 命令行参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--taxi-type` | str | green | 出租车类型：green 或 yellow |
| `--file-path` | str | - | Parquet 数据文件路径（可选，默认自动选择） |
| `--topic` | str | - | Kafka 主题名称（可选，默认自动选择） |
| `--rate` | int | 10 | 发送速率（条/秒） |
| `--start-offset` | int | 0 | 数据文件起始位置 |

#### 3.1.2 核心流程

```python
def main():
    # 1. 解析命令行参数
    args = parse_args()
    
    # 2. 加载配置
    load_dotenv()
    config = load_config(args.taxi_type)
    
    # 3. 读取检查点（断点续传）
    start_offset = load_checkpoint(config['topic']) or args.start_offset
    
    # 4. 读取 Parquet 文件
    df = pd.read_parquet(config['file_path'])
    
    # 5. 初始化 Kafka 生产者
    producer = KafkaProducer(
        bootstrap_servers=config['kafka_servers'],
        value_serializer=lambda v: json.dumps(v).encode('utf-8')
    )
    
    # 6. 循环发送消息
    for i, row in enumerate(df.iloc[start_offset:].itertuples(), start=start_offset):
        # 数据校验
        if validate_record(row):
            # 发送消息
            message = row._asdict()
            producer.send(config['topic'], value=message)
            
            # 速率控制
            time.sleep(1 / args.rate)
            
            # 定期保存检查点
            if i % 100 == 0:
                save_checkpoint(config['topic'], i)
                logger.info(f"Sent {i} records")
    
    # 7. 完成，清理检查点
    producer.flush()
    clear_checkpoint(config['topic'])
    logger.info("All records sent successfully")
```

### 3.2 数据校验机制

#### 3.2.1 校验规则

| 字段 | 校验条件 | 说明 |
|------|---------|------|
| VendorID | not null | 供应商ID不能为空 |
| pickup_datetime | not null | 上车时间不能为空 |
| dropoff_datetime | not null | 下车时间不能为空 |
| passenger_count | not null | 乘客数不能为空 |
| trip_distance | not null and &gt; 0 | 行程距离必须为正 |
| PULocationID | not null | 上车区域ID不能为空 |
| DOLocationID | not null | 下车区域ID不能为空 |
| total_amount | not null and &gt; 0 | 总金额必须为正 |

#### 3.2.2 校验实现

```python
def validate_record(row):
    """
    校验单条记录的有效性
    """
    # 非空校验
    required_fields = [
        'VendorID', 'pickup_datetime', 'dropoff_datetime',
        'passenger_count', 'trip_distance',
        'PULocationID', 'DOLocationID', 'total_amount'
    ]
    
    for field in required_fields:
        if pd.isna(getattr(row, field)):
            logger.warning(f"Record missing required field: {field}")
            return False
    
    # 值域校验
    if row.trip_distance &lt;= 0:
        logger.warning(f"Invalid trip_distance: {row.trip_distance}")
        return False
    
    if row.total_amount &lt;= 0:
        logger.warning(f"Invalid total_amount: {row.total_amount}")
        return False
    
    # 时间逻辑校验
    if row.dropoff_datetime &lt;= row.pickup_datetime:
        logger.warning("Dropoff time must be after pickup time")
        return False
    
    return True
```

```python
def load_checkpoint(topic):
    """加载检查点"""
    checkpoint_file = f"checkpoint/{topic}_checkpoint.json"
    if os.path.exists(checkpoint_file):
        with open(checkpoint_file, 'r') as f:
            data = json.load(f)
            return data['offset']
    return None

def save_checkpoint(topic, offset, total_records=None, sent_records=None):
    """保存检查点"""
    checkpoint_file = f"checkpoint/{topic}_checkpoint.json"
    os.makedirs("checkpoint", exist_ok=True)
    data = {
        "offset": offset,
        "timestamp": int(time.time()),
        "total_records": total_records,
        "sent_records": sent_records
    }
    with open(checkpoint_file, 'w') as f:
        json.dump(data, f)

def clear_checkpoint(topic):
    """清除检查点（发送完成时）"""
    checkpoint_file = f"checkpoint/{topic}_checkpoint.json"
    if os.path.exists(checkpoint_file):
        os.remove(checkpoint_file)
```

### 3.4 速率控制实现

#### 3.4.1 简单速率控制

```python
def rate_limited_send(producer, topic, messages, rate):
    """
    速率控制的消息发送
    rate: 每秒发送的消息数
    """
    interval = 1.0 / rate  # 每条消息的间隔时间
    
    for message in messages:
        start_time = time.time()
        
        # 发送消息
        producer.send(topic, value=message)
        
        # 计算消耗的时间，补眠
        elapsed = time.time() - start_time
        if elapsed &lt; interval:
            time.sleep(interval - elapsed)
```

#### 3.4.2 批量发送优化

```python
def batch_send_with_rate(producer, topic, messages, rate, batch_size=10):
    """
    批量发送 + 速率控制
    """
    interval = batch_size / rate  # 每批的间隔时间
    
    for i in range(0, len(messages), batch_size):
        start_time = time.time()
        
        # 批量发送
        batch = messages[i:i+batch_size]
        for message in batch:
            producer.send(topic, value=message)
        
        # 刷写确认
        producer.flush()
        
        # 速率控制
        elapsed = time.time() - start_time
        if elapsed &lt; interval:
            time.sleep(interval - elapsed)
```

### 3.5 配置管理 - kafka_config.py

#### 3.5.1 配置结构

```python
def load_config(taxi_type):
    """
    根据出租车类型加载配置
    """
    base_config = {
        'kafka_servers': os.getenv('KAFKA_BOOTSTRAP_SERVERS', 
                                    'localhost:9092'),
    }
    
    if taxi_type == 'green':
        return {
            **base_config,
            'topic': os.getenv('KAFKA_TOPIC_GREEN', 'taxi_trip_green'),
            'file_path': os.getenv('GREEN_TRIPDATA_PATH', 
                                    'data/green_tripdata_2025-04.parquet')
        }
    elif taxi_type == 'yellow':
        return {
            **base_config,
            'topic': os.getenv('KAFKA_TOPIC_YELLOW', 'taxi_trip_yellow'),
            'file_path': os.getenv('YELLOW_TRIPDATA_PATH', 
                                    'data/yellow_tripdata_2025-04.parquet')
        }
    else:
        raise ValueError(f"Unknown taxi type: {taxi_type}")
```
