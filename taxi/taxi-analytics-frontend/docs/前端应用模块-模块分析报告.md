
# Taxi Analytics Frontend 模块分析报告

## 1. 模块概述

Taxi Analytics Frontend 是纽约市出租车数据分析系统的**前端应用模块**，基于 Vue 3 + TypeScript + Vite 构建，提供直观的数据可视化和交互界面。

### 1.1 核心功能

| 功能模块 | 描述 | 状态 |
|---------|------|------|
| **Dashboard** | 核心 KPI 展示、趋势图、分布图 | ✅ |
| **Analysis** | 多维度分析（小时/周几/供应商/支付/机场等） | ✅ |
| **Realtime** | 实时指标、热点区域、费用构成 | ✅ |
| **Map** | 热力图可视化、区域分析 | ✅ |
| **Quality** | 数据质量监控、告警信息 | ✅ |
| **AI** | 智能对话查询、自然语言转图表 | ✅ |
| **Export** | 数据导出功能 | ✅ |

### 1.2 技术栈

| 技术组件 | 版本 | 用途 |
|---------|------|------|
| Vue | 3.4+ | 前端框架 |
| TypeScript | 5.0+ | 类型系统 |
| Vite | 5.0+ | 构建工具 |
| ECharts | 5.4+ | 图表库 |
| Pinia | 2.1+ | 状态管理 |
| Vue Router | 4.2+ | 路由管理 |
| Axios | 1.6+ | HTTP 客户端 |
| @taxi-analytics/shared-types | 1.0+ | 共享类型定义 |

### 3.2 API 请求封装

```typescript
// src/api/request.ts
import axios, { AxiosInstance, AxiosRequestConfig, AxiosResponse } from 'axios'
import type { Result } from './types'

const request: AxiosInstance = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || 'http://localhost:8080/api',
  timeout: 30000
})

// 请求拦截器
request.interceptors.request.use(
  (config) =&gt; {
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  (error) =&gt; Promise.reject(error)
)

// 响应拦截器
request.interceptors.response.use(
  (response: AxiosResponse&lt;Result&gt;) =&gt; {
    const { code, message, data } = response.data
    if (code === 200) {
      return data
    } else {
      console.error(message)
      return Promise.reject(new Error(message))
    }
  },
  (error) =&gt; Promise.reject(error)
)

export default request
```

```vue
  // 添加用户消息

      role: 'assistant',

      data: response.data,
      chartConfig: response.chartConfig,

    })
  } catch (error) {
    console.error('AI 查询失败:', error)
  } finally {
    loading.value = false
```

### 3.4 图表渲染组件

```vue
&lt;!-- src/components/ai/ResultViewer.vue --&gt;
&lt;template&gt;
  &lt;div class="result-viewer"&gt;
    &lt;div ref="chartRef" class="chart-container"&gt;&lt;/div&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script setup lang="ts"&gt;
import { ref, onMounted, watch } from 'vue'
import * as echarts from 'echarts'
import type { ChartConfig } from '@taxi-analytics/shared-types'

const props = defineProps&lt;{
  config: ChartConfig
  data: any[]
}&gt;()

const chartRef = ref&lt;HTMLElement&gt;()
let chart: echarts.ECharts | null = null

const initChart = () =&gt; {
  if (!chartRef.value) return
  
  chart = echarts.init(chartRef.value)
  renderChart()
  
  window.addEventListener('resize', () =&gt; chart?.resize())
}

const renderChart = () =&gt; {
  if (!chart || !props.config) return
  
  const option = getChartOption()
  chart.setOption(option)
}

const getChartOption = () =&gt; {
  const { chartType, xAxisField, yAxisField, title } = props.config
  const xData = props.data.map(d =&gt; d[xAxisField])
  const yData = props.data.map(d =&gt; d[yAxisField])
  
  if (chartType === 'line') {
    return {
      title: { text: title },
      xAxis: { type: 'category', data: xData },
      yAxis: { type: 'value' },
      series: [{
        type: 'line',
        data: yData,
        smooth: true,
        areaStyle: { color: 'rgba(59, 130, 246, 0.3)' }
      }]
    }
  } else if (chartType === 'bar') {
    return {
      title: { text: title },
      xAxis: { type: 'category', data: xData },
      yAxis: { type: 'value' },
      series: [{
        type: 'bar',
        data: yData,
        itemStyle: { color: '#3b82f6' }
      }]
    }
  } else if (chartType === 'pie') {
    return {
      title: { text: title },
      series: [{
        type: 'pie',
        data: props.data.map(d =&gt; ({
          name: d[xAxisField],
          value: d[yAxisField]
        }))
      }]
    }
  }
}

watch(() =&gt; [props.config, props.data], () =&gt; {
  renderChart()
})

onMounted(() =&gt; {
  initChart()
})
&lt;/script&gt;
```

```vue
    &lt;/div&gt;
  &lt;/div&gt;
&lt;/template&gt;

&lt;script setup lang="ts"&gt;
import { ref, onMounted, onUnmounted } from 'vue'
import * as echarts from 'echarts'
import { getRealtimeKpi, getHotspots, getFeeComposition } from '@/api/realtime'
import type { RealtimeKpiDTO, HotspotDTO, FeeCompositionDTO } from '@taxi-analytics/shared-types'

const kpi = ref&lt;RealtimeKpiDTO | null&gt;(null)
const hotspots = ref&lt;HotspotDTO[]&gt;([])
const feeComposition = ref&lt;FeeCompositionDTO[]&gt;([])

const feeChartRef = ref&lt;HTMLElement&gt;()
let feeChart: echarts.ECharts | null = null

// 轮询获取数据
let intervalId: number | null = null

const fetchData = async () =&gt; {
  try {
    kpi.value = await getRealtimeKpi()
    hotspots.value = await getHotspots()
    feeComposition.value = await getFeeComposition()
    renderFeeChart()
  } catch (error) {
    console.error('获取实时数据失败:', error)
  }
}

const renderFeeChart = () =&gt; {
  if (!feeChartRef.value || feeComposition.value.length === 0) return
  
  if (!feeChart) {
    feeChart = echarts.init(feeChartRef.value)
  }
  
  feeChart.setOption({
    series: [{
      type: 'pie',
      data: feeComposition.value.map(f =&gt; ({
        name: f.paymentType,
        value: f.totalAmount
      }))
    }]
  })
}

onMounted(() =&gt; {
  fetchData()
  intervalId = window.setInterval(fetchData, 5000)
})

onUnmounted(() =&gt; {
  if (intervalId) clearInterval(intervalId)
  feeChart?.dispose()
})
&lt;/script&gt;
```

### 3.6 状态管理 (Pinia)

```typescript
// src/stores/dashboard.ts
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import { getKpiSummary, getTrendData } from '@/api/dashboard'
import type { KpiSummaryDTO, TrendDataDTO } from '@taxi-analytics/shared-types'

export const useDashboardStore = defineStore('dashboard', () =&gt; {
  const kpiSummary = ref&lt;KpiSummaryDTO | null&gt;(null)
  const trendData = ref&lt;TrendDataDTO[]&gt;([])
  const loading = ref(false)
  
  const fetchKpiSummary = async () =&gt; {
    loading.value = true
    try {
      kpiSummary.value = await getKpiSummary()
    } finally {
      loading.value = false
    }
  }
  
  const fetchTrendData = async () =&gt; {
    loading.value = true
    try {
      trendData.value = await getTrendData()
    } finally {
      loading.value = false
    }
  }
  
  const totalRevenue = computed(() =&gt; kpiSummary.value?.totalRevenue || 0)
  const totalTrips = computed(() =&gt; kpiSummary.value?.totalTrips || 0)
  
  return {
    kpiSummary,
    trendData,
    loading,
    totalRevenue,
    totalTrips,
    fetchKpiSummary,
    fetchTrendData
  }
})
```
