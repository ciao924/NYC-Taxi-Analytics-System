# 前端应用模块

基于 Vue 3 的数据可视化前端应用。

## 📌 模块定位

本模块是系统的**用户界面层**，提供数据可视化、实时监控、智能查询等交互功能。

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| **数据仪表盘** | 核心指标可视化展示 |
| **实时监控** | 实时数据指标监控 |
| **多维分析** | 多维度数据钻取分析 |
| **热力地图** | 地理数据可视化 |
| **质量监控** | 数据质量仪表盘 |
| **AI 智能查询** | 自然语言查询助手 |

## 🛠️ 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue.js | 3.4+ | 前端框架 |
| Element Plus | 2.6+ | UI 组件库 |
| ECharts | 5.5+ | 图表库 |
| TypeScript | 5.4+ | 类型安全 |
| Vite | 5.2+ | 构建工具 |

## 🚀 运行方式

### 安装依赖

```bash
cd taxi/taxi-analytics-frontend
npm install
```

### 开发模式

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```
