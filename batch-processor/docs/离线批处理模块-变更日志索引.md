# 离线批处理模块 - 变更日志

## 模块说明

本模块负责离线数据处理，包括：
- ODS 层：Parquet 到 Hive 表
- DWD 层：数据清洗和维度构建
- DWS 层：聚合汇总
- ADS 层：指标输出到 MySQL
- 数据质量检测
